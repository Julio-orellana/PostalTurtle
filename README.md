# Proyecto 2, Postal Con Turtle .
Diseño de postal para incentivar el Turismo en Guatemala w Marco Díaz, dibujo de paisaje en un canvas creado especialmente para la elaboracion del programa, bucle para ejecutar un menu siempre que el usuario lo desee, opcion a ingresar de quien y para quien esta dirigida la postal, utilizacion de funciones, metodos, librerias, modulos, variables, inputs en nuestro programa.


Documentacion a detalle del programa y cada funcion dentro del archivo "modulos".


Ejemplo de uso:
